# 관리자 및 개발 환경 가이드 (MANAGER_GUIDE.md)

## 1. 운영 환경 구축 (Environment Setup)

### 시스템 필수 프로그램
| 필요 컴포넌트 | 요구 버전 | 설치 목적 |
| --- | --- | --- |
| **Python** | 3.x 이상 권장 | 로컬 정적 웹 서버(`http.server`)를 백그라운드로 띄우기 위한 런타임 |
| **Browser** | Chromium 엔진 최신 구동기 | ES6 문법 및 HTML5 Canvas 요소의 원활한 그래픽 가속 테스트 |

### 숨김 변수 및 하위 의존성 제로화 정책
해당 프로젝트는 순수 바닐라 환경을 절대적으로 치향하므로 `npm` 패키지(node_modules)나 `.env` 파일과 같은 별도의 복잡한 종속성 관리 파일이 포함되지 않습니다. 보안상 감춰야 할 토큰 또한 일체 존재하지 않습니다.

## 2. 서버 빌드 및 로컬 서비스 론칭

프론트엔드 정적 파일이지만, 자바스크립트의 모듈 불러오기 등 엄격한 브라우저 자체 보안 정책(CORS)을 준수하기 위해 로컬 컴퓨터 상에 가상 호스트 서버를 올려 서비스하는 것을 원칙으로 삼습니다.

### 로컬 호스팅 자동화 스크립트 실행
1. 프로젝트 최상위 루트 디렉토리에서 PowerShell 등을 엽니다.
2. `.\.scripts\run-local.ps1` 명령을 실행합니다.

| 실행 절차 파이프라인 | 성공 시 콘솔 출력 결과 시나리오 | 오류 시 대체 및 조치 방안 |
| --- | --- | --- |
| 1. Python 런타임 바이너리 탐색 | `✅ Python 실행기 확인됨` 메시지 출력 | Windows 환경 변수(PATH)에 Python 수동 등록 요망 |
| 2. 기본 서비스 포트(80) 장악 시도 | `로컬 서버를 포트 80 에서 시작합니다` | (일반적으로 Administrator 권한 필요) |
| 3. 서비스 포트 우회(8080) 안전 폴백 | 포트 80 실패 시 **포트 8080**으로 자동 우회 할당 | 브라우저에 `http://localhost:8080` 수동 타이핑하여 진입 |

서버 프로세스의 강제 종료를 위해서는 프로세스가 상주 중인 터미널 창에서 물리적인 인터럽트 단축키(`Ctrl + C`)를 전송합니다.
