# Neon Game World 사용자 매뉴얼 (USER_GUIDE.md)

## 📋 핵심 서비스 이용 흐름 시나리오

### 1초 만에 진입하는 오락실 로비
관리자가 터미널에서 스크립트 기동을 마친 뒤 브라우저를 열면, 즉시 환상적인 글래스 기반의 메인 로비가 반겨줍니다! 3종의 투명한 게임 카드에 마우스를 천천히 올려(Hover) 반짝이는 고유 네온 이펙트 렌더링을 확인해 보세요. 마음이 이끌리는 게임 카드를 마우스로 클릭하면 즉시 해당 모드의 캔버스로 다크 스위칭됩니다.

### 아케이드 공통 조작 키
| 키보드/마우스 입력 | 수행 명령 및 동작 | 화면 내 적용 범위 |
| --- | --- | --- |
| `ESC` | 화면 중앙에 떠 있는 게임별 모달형 도움말(Help) 시트를 닫아버립니다. | 전역 시스템 (Global) |
| **좌상단 게임 패드 아이콘 클릭** | 현재 진행 중인 게임의 로직을 일시 중단 및 파기한 채 메인 코어 로비 화면으로 탈주합니다. | 전역 시스템 (Global) |

## 🎮 미니 게임별 상세 조작 타격 가이드

### 테트리스 (Tetris)
| 입력 콘솔 키 | 인게임 기능 | 전략 요점 가이드 |
| --- | --- | --- |
| `↑` (위 방향키) | 떨어지는 도형(테트로미노) 회전 | 하단 지형지물에 맞춰 재빠르게 모델링의 모양새를 돌려놓습니다. |
| `←` / `→` | 블록 렌더링 위치 좌/우측 이동 | - |
| `↓` (아래 방향키) | 드롭 속도 가속화 (소프트 드롭) | 빈칸을 조준한 후 재빠르게 아래로 하강시킵니다. |
| `스페이스(Space)` | 하드 드롭 (바닥에 즉시 강타) | 망설임 없이 바닥의 목표 지점으로 순식간에 쳐박아버립니다. |

### 스네이크 (Snake)
| 입력 콘솔 키 | 인게임 기능 | 전략 요점 가이드 |
| --- | --- | --- |
| `←` `↑` `→` `↓` | 플레이어의 바라보는 이동 방향 90도 전향 | 머리가 나아가던 궤적의 완전 반대 방향(예: 위쪽 직진 중 아래 화살표 입력)은 구조적으로 무시됩니다! |
| **목표 조준** | 화면에 스폰되어 점멸하는 붉은 먹이를 물리적으로 타격하여 먹방 | 먹을 때마다 몸통 체인이 한 마디씩 길어지고 가속도가 붙습니다. 늘어난 자신의 꼬리를 조심하세요! |

### 슈팅 디펜스 (Shooter)
| 입력 콘솔 키 | 인게임 기능 | 전략 요점 가이드 |
| --- | --- | --- |
| `←` / `→` | 플레이어의 비행 기체 회피 기동 | 전형적인 갤러그 스타일로 쏟아지는 적기의 포화를 요리조리 회피합니다. |
| `스페이스(Space)` | 단발성 미사일 발사 (누름 유지 시 다중 연사) | 위에서 하강하며 덮쳐오는 적 함대에 즉각적인 피해를 가해 완전히 섬멸시킵니다! |

## ⚠️ 경고 및 플레이 주의사항
* Web Audio API 합성 모듈을 사용하여 꽤 큰 소음 주파수가 발생할 수 있으므로, 게임 시작 전 운영체제 스피커 볼륨을 확인해 주세요.
* 모바일 기기의 터치스크린 보다는 완전한 조이패드 혹은 키보드 환경에서의 플레이를 상정하고 제작되었습니다.
