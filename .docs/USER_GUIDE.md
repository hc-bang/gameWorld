# 🎮 사용자 가이드 (User Guide)

> 게임 월드에 오신 것을 환영합니다!  
> 이 문서에서는 게임 포털의 기본 사용 방법과 각 게임의 규칙을 안내합니다.

---

## 🌟 시작하기

### 접속 방법
- **웹 버전**: 브라우저에서 `https://hc-bang.github.io/gameWorld/` 에 접속합니다.
- **로컬 버전**: 로컬 서버 실행 후 `http://localhost` 에 접속합니다.

### 메인 메뉴에서의 게임 선택
메인 화면에서 원하는 게임 카드를 클릭하면 해당 게임으로 이동합니다.

| 카드 | 게임 | 난이도 |
|------|------|--------|
| 🧱 테트리스 | 클래식 퍼즐 | ⭐⭐ 중간 |
| 🐍 스네이크 | 방향 조작 | ⭐⭐ 중간 |
| 🚀 갤러그 | 슈팅 액션 | ⭐⭐⭐ 어려움 |

## 🏠 공통 기능

### 홈 버튼
- 게임 화면 **좌측 상단**의 🏠 버튼을 클릭하면 언제든지 메인 메뉴로 돌아갈 수 있습니다.
- 메인 메뉴로 돌아가면 현재 게임의 진행 상황은 초기화됩니다.

### HELP 버튼
- 게임 중 우측의 **HELP** 버튼을 클릭하면 조작키 안내 모달이 열립니다.
- 모달이 열리면 게임이 **일시 정지**됩니다.
- 모달 닫기: **GOT IT!** 버튼, 우측 상단 **×** 버튼, 또는 **모달 외부 클릭**, **ESC 키**로 닫을 수 있습니다.
- 모달을 닫으면 게임이 자동으로 **재개**됩니다.

### 메뉴 복귀 버튼 (게임 오버 시)
- 게임 오버 화면에서 **MENU** 버튼을 클릭하면 메인 메뉴로 돌아갑니다.

## 🧱 테트리스 (Tetris)

### 목표
떨어지는 블록을 쌓아 가로 줄(라인)을 꽉 채우면 해당 줄이 사라지고 점수를 얻습니다. 블록이 화면 위쪽까지 쌓이면 게임이 종료됩니다.

### 조작 방법

| 키 | 동작 |
|----|------|
| `←` (왼쪽 방향키) | 블록을 왼쪽으로 이동 |
| `→` (오른쪽 방향키) | 블록을 오른쪽으로 이동 |
| `↑` (위쪽 방향키) | 블록을 시계방향으로 회전 |
| `↓` (아래쪽 방향키) | 블록을 한 칸씩 천천히 하강 (소프트 드롭) |
| `Space` (스페이스바) | 블록을 즉시 착지 (하드 드롭) |

### 점수 계산

| 클리어한 라인 수 | 획득 점수 |
|-----------------|-----------|
| 1줄 | 100 × 레벨 |
| 2줄 | 200 × 레벨 |
| 3줄 | 300 × 레벨 |
| 4줄 (테트리스!) | 400 × 레벨 |

### 레벨 및 난이도
- **10라인** 제거마다 레벨이 1 올라갑니다.
- 레벨이 올라갈수록 블록이 더 빠르게 떨어집니다.

### 특수 기능
- **고스트 블록**: 현재 블록이 어디에 떨어질지 반투명 윤곽선으로 미리 보여줍니다.
- **NEXT 패널**: 다음에 나올 블록을 미리 확인할 수 있습니다.
- **파티클 이펙트**: 라인 클리어 시 색상 파티클이 분출됩니다.

### 블록 종류 및 색상

| 블록 | 색상 |
|------|------|
| I (일자 막대) | 청록색 |
| J (J자형) | 파란색 |
| L (L자형) | 주황색 |
| O (정사각형) | 노란색 |
| S (S자형) | 초록색 |
| T (T자형) | 보라색 |
| Z (Z자형) | 빨간색 |

## 🐍 스네이크 (Snake)

### 목표
방향키로 뱀을 조종하여 화면에 나타나는 빨간 먹이를 먹습니다. 먹이를 먹을수록 뱀이 길어지고 점수가 올라갑니다. 벽이나 자신의 몸에 부딪히면 게임 오버입니다.

### 조작 방법

| 키 | 동작 |
|----|------|
| `←` (왼쪽 방향키) | 왼쪽으로 방향 전환 |
| `→` (오른쪽 방향키) | 오른쪽으로 방향 전환 |
| `↑` (위쪽 방향키) | 위쪽으로 방향 전환 |
| `↓` (아래쪽 방향키) | 아래쪽으로 방향 전환 |

> **주의**: 진행 방향의 반대로는 즉시 전환할 수 없습니다. (예: 오른쪽으로 이동 중 왼쪽 키 무시)

### 점수 계산

| 이벤트 | 획득 점수 |
|--------|-----------|
| 먹이 1개 획득 | 10점 |

### 레벨 및 난이도
- **50점**마다 레벨이 1 올라갑니다.
- 레벨이 올라갈수록 뱀의 이동 속도가 빨라집니다.

### 게임 오버 조건
- 뱀의 머리가 **벽에 충돌**하는 경우
- 뱀의 머리가 **자신의 몸에 충돌**하는 경우

### 특수 기능
- **카운트다운**: 게임 시작 전 3초 카운트다운이 진행됩니다.
- **파티클 이펙트**: 먹이 획득 시 파티클 이펙트가 발생합니다.
- **격자 배경**: 반투명 격자로 이동 단위를 직관적으로 파악할 수 있습니다.

## 🚀 갤러그 (Galaga)

### 목표
화면 하단의 우주선을 조종하여 적 기체들을 모두 격파합니다. 적의 공격에 피격당하면 HP가 줄어들며, HP가 0이 되면 게임 오버입니다. 적을 모두 제거하면 다음 웨이브가 시작됩니다.

### 조작 방법

| 키 | 동작 |
|----|------|
| `←` (왼쪽 방향키) | 우주선을 왼쪽으로 이동 |
| `→` (오른쪽 방향키) | 우주선을 오른쪽으로 이동 |
| `Space` (스페이스바) | 미사일 발사 (200ms 간격 연사 가능) |

### 점수 계산

| 적 종류 | 모양 | 색상 | 점수 |
|---------|------|------|------|
| 하위 적 (Drone) | 다이아몬드 ◇ | 빨간색 | 10점 |
| 중위 적 (Fighter) | 정사각형 □ | 분홍색 | 20점 |
| 상위 적 (Elite) | 삼각형 △ | 초록색 | 30점 |

### HP 시스템
- 플레이어는 총 **3 HP** (❤️ ❤️ ❤️)를 보유합니다.
- 적의 총알에 피격되거나, 다이브 중인 적 기체와 충돌하면 HP가 1 감소합니다.
- HP가 0이 되면 게임 오버입니다.

### 적 행동 패턴

| 패턴 | 설명 |
|------|------|
| **대형 유지** | 적들이 V자 형태로 좌우로 천천히 이동합니다. |
| **다이브 공격** | 주기적으로 무작위 적 1기가 대형에서 이탈해 플레이어를 향해 돌진합니다. |
| **복귀** | 다이브 후 화면 밖으로 나간 적은 상단에서 재등장해 대형으로 복귀합니다. |
| **사격** | 대형 유지 중인 적들이 주기적으로 아래쪽으로 총알을 발사합니다. |

### 레벨 및 난이도
- **500점**마다 레벨이 올라갑니다.
- 레벨이 올라갈수록:
  - 대형 이동 속도 증가
  - 다이브 공격 주기 단축
  - 적 총알 속도 증가

## ❓ FAQ

**Q. 게임 중에 소리가 나지 않아요.**  
A. 브라우저의 오디오 정책으로 인해 첫 클릭 전에는 소리가 재생되지 않을 수 있습니다. 게임 카드를 클릭하면 자동으로 오디오가 활성화됩니다.

**Q. 모바일에서도 플레이할 수 있나요?**  
A. 현재 버전은 키보드 조작을 기준으로 제작되어 있으며, 모바일 터치 조작은 지원되지 않습니다.

**Q. 점수가 저장되나요?**  
A. 현재 버전은 점수 저장 기능을 제공하지 않습니다. 각 세션의 최종 점수만 게임 오버 화면에서 확인할 수 있습니다.

**Q. 갤러그에서 적이 끝없이 나와요.**  
A. 모든 적을 격파하면 새로운 웨이브가 자동으로 생성됩니다. 무한 웨이브 방식으로 설계되었습니다.
