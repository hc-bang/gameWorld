# 💻 시스템 코드 규격 및 자산 (PROJECT_CODE)

> 프로젝트 내 구동되는 스크립트와 객체 논리, 식별된 기술적 자산과 미래 리팩토링 필요 항목을 기록합니다.

---

## 📌 주요 클래스 및 기능

- **`GameManager` (main.js):** 화면 전이 상태 제어, 게임 생명주기(start/stop/restart) 조정, 입력 리스너 바인딩 지원
- **`SoundManager` (main.js):** Web Audio API 기반 오실레이터를 거쳐 주파수(Frequency) 단위 게임용 이펙트음 즉시 생성
- **`ParticleSystem` (main.js):** 캔버스용 풀스크린 배경 파티클 유지 로직
- **`Tetris` (tetris.js):** 메인 보드 배열 처리, 키보드 조작 대응, 고스트 피스 역산 및 바닥 충돌/병합 로직
- **`Snake` (snake.js):** 세그먼트형 자료구조 배열(`Array.prototype.unshift()`) 활용 뱀 움직임 처리, 카운트다운 타이머 병합
- **`Shooter` (shooter.js):** 객체 렌더링 배열(Bullet, Enemy) 구성, 다이브 AI 패턴 및 대형 회귀 로직 적용

## 🚨 잠재적 기술 부채 및 개선안

본 항목은 시스템 현행화 중 파편화된 공통 기능으로써 향후 성능 향상 및 체적 관리를 위해 통합 병합 검토 대상으로 확인된 로직입니다.

| 항목 | 내용 | 리팩토링 제안 사항 |
| --- | --- | --- |
| **파티클 렌더링 파편화** | `Tetris`, `Shooter`, `Snake` 내 각각 독립적인 `createParticles(x, y, color)` 및 캔버스 업데이트 루틴 분리 존재. | 단일 `FXManager`를 구성해 코어 렌더링 및 배열을 전역에서 제어. **단, 현행 버전에서는 개별 독립 캔버스가 존재해 안정성을 1순위로 고려하여 임의 분리시키지 않고 원형 유지 중.** |
